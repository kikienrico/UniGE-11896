### Funzioni Pt.2
- **Parametro passato per valore** → La funzione riceve una copia del contenuto corrispondente al parametro attuale 
``` C++
void funzione(int par)
return par++; //viene ritornata una copia di par
```
- **Passaggio di parametro per riferimento** →  La funzione riceve l’indirizzo di memoria (la locazione) del parametro corrispondente attuale, si segna con &. Il parametro viene modificato direttamente.
- Se ho un vector di dimensioni enormi, invece di leggerlo creando una copia (*passaggio per valore*), lo vado a leggere direttamente nella sua cella di memoria (*per risparmiare risorse, e velocizzare l’esecuzione*).
``` C++
void funzione (int& i)
```

### Funzione Swap
- **Senza funzione swap.**
``` C++
void swap (int& x, int& y) {
	int tmp;
	tmp=x; x=y; y=tmp;
}
```
- **Con funzione swap**
``` C++
int main() {
int a, b;
swap(a, b);
}
```

### Passaggio per riferimento costante
``` C++
float media(const vector<float> &v) {
	float somma=0;
	for (int i=0; i<v.size(); i++)
		somma+=v[i];
	return somma/v.size();
}
```
> Effettuando il passaggio di riferimento costante, mantengo l’efficienza della lettura di memoria, ma **non** vado a modificare il valore dei parametri attuali

### Parametri
- **IN** → valori destri che la funzione prende in input, il cui valore serve per svolgere operazioni all’interno della funzione stessa (*per valore/per riferimento costante*)
- **OUT** → valori sinistri che la funzione produce in output, modificando i relativi parametri attuali (effetto collaterale, diverso dal valore prodotto dalla funzione col return) (*per riferimento*)
- **IN-OUT** → valori destri e sinistri, che la funzione prende in input e che può modificare in output (*per riferimento*)
