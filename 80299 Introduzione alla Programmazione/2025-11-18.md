### Stream (sequenze di caratteri)
**Operazioni degli stream:**
- *Input* → leggere (e interpretare) input formattato
- *Output* → formattare (trasformo in rappresentazione ASCII) e scrivere informazione
> [!info] 
>  Si chiamano stream perché sono “ruscelli” d’informazione che scorrono, man mano che le utilizziamo “le consumiamo” e finiscono fuori dallo stream. (Vengono perse).

**Tipi di stream:**
- *cin*
- *cout*
- *File (ASCII)* → contenitore di informazioni (caratteri) su disco
	- Si possono associare a `STREAM`
		1. Apertura file (associa variabile `STREAM` a nome file). (Posso aprirli in *Input* o *Output*)
			- *Input* → Leggo dal file, per poi utilizzare i dati nel programma
			- *Output* → Scrivo nel file, attraverso il programma
		2. Operazioni di lettura (*Input*) o scrittura (*Output*) sequenziale
		3. Chiusura stream: “sgancia” le variabili dal file 

> [!important] 
>  **Buffer**
>  È un allocazione di memoria (il ruscello, stream) che viene riempito di informazioni veloci, e quando viene deciso dal *runtime environment*, il buffer viene svuotato e fa spazio ad altre informazioni.

### Sintassi
`#include <iostream>` → necessario per usare gli stream (cin, cout)
`#include <fstream>` → necessario per usare i files *input/output*

**Dichiarazione di variabile stream:**
- `tipo` nomeVariabile;
- `istream` fileInput;
- `ostream` fileOutput;
### Apertura
> nomeFile o è una string o è un literal. 

- **Input**
	**String** → `fileInput.open("nomeFile.txt")`
	**Literal** → `fileInput.open(varString)` con variabile string (`string varString="nomeFile.txt"`)
- **Output**
	**String** → `fileOutput.open("nomeFile.txt")`
	**Literal** → `fileOutput.open(varString)` con variabile string (`string varString="nomeFile.txt"`)

Per scrivere su un `ostream`:
	`fileOutput << output formattato`
Per leggere da un `istream`:
	`fileInput >> variabile` (variabile del tipo previsto dal programma, concatenazione mediante >>)

### Controllo errori
``` Cpp
// Errori:
if(!fileInput) //sto verificando se fileInput è in stato di errore

string s="myFile.txt";
istream fileInput(s);
if(!fileInput) {
	cout << "Il file " << s << " non è accessibile!\n"
	return;
}
```
### Chiusura e fine file
- **Fine file** → `fileInput.eof()` → funzione Boolean che vale true dopo che *abbiamo tentato di leggere a file finito*.
- **Chiusura file** → `fileInput.close()` e `fileOutput.close()` 

> [!important] 
>  eof → end of file
