### Struct
> Permette di aggregare i **dati non omogenei**
``` C++
//esempio
struct prodotto {
	string descrizione;
	string categoria;
	double prezzo;
};

```

**Come accedere ai campi struct?**
> Per accedere ai campi struct si utilizza la **dot notation**
``` C++
//esempio
struct prodotto {
	string descrizione;
	string categoria;
	double prezzo;
};

prodotto.descrizione;
prodotto.categoria;
prodotto.prezzo;
```
