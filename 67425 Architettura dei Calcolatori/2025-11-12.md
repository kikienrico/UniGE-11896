### Funzioni in RISC-V
``` cpp
int sum (int x, int y) {
return x+y;
}

int main() {
	int a, 6;
	a = sum(3, 4);
	a = a << 2;
	b = sum(a, a);
	b = b*4;
}
```

> Per chiamare una funzione si usa JAL (Jump and Link), il formato è `JAL etichetta + RET`, Return torna indietro all’indirizzo.

### Registri
- **x1, x2** (ra → Return Address, sp → Stack Pointer)
- **t0-t6** → registri temporanei, ogni funzione può usarli
- **s0-s11** → ogni funzione può usarli ma solo dopo che si ha salvato il suo valore
- **a0-a7**
